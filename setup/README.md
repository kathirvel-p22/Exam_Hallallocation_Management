# Exam Seat Allocation Management System - Setup Guide

Welcome to the Exam Seat Allocation Management System setup guide! This directory contains all the tools and documentation you need to get the system running on your local machine.

## Quick Start

For a fast setup, run the quick start script:

```bash
php setup/quick_start.php
```

This will automatically check your system requirements, set up the database, and configure the application.

## Manual Setup

If you prefer a manual setup or need more control, follow the [Setup Guide](setup_guide.md) for detailed step-by-step instructions.

## What's Included

- **setup_guide.md** - Comprehensive installation and configuration guide
- **quick_start.php** - Automated setup script for beginners
- **local_config.php** - Development configuration settings
- **.htaccess** - Web server configuration for Apache
- **deployment_checklist.php** - Validation script to ensure everything works
- **requirements.txt** - System requirements and dependencies

## Support

- Works with XAMPP, WAMP, and MAMP
- Compatible with Apache and Nginx
- Beginner-friendly with detailed troubleshooting
- Includes sample data for testing

## Next Steps

1. Check the [requirements.txt](requirements.txt) to ensure your system is compatible
2. Run the [quick_start.php](quick_start.php) script or follow the [setup_guide.md](setup_guide.md)
3. Use the [deployment_checklist.php](deployment_checklist.php) to validate your setup
4. Start developing or using the system!

## Getting Help

If you encounter issues during setup:

1. Check the troubleshooting section in [setup_guide.md](setup_guide.md)
2. Run the [deployment_checklist.php](deployment_checklist.php) for diagnostics
3. Review the [requirements.txt](requirements.txt) for compatibility
